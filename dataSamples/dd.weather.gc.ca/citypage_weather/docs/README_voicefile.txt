###     About      ###
######################

This document explains voices files in mp3 format available
on the datamart of Environment Canada under the following URL:
http://dd.weather.gc.ca/citypage_weather/mp3/

Each file allows the user to listen the weather forecast for
various cities of Canada or the current weather conditions.

There are 3 types of files:
1- File with the current weather conditions
2- File with the short term forecast (Day 1 and 2)
3- File with the outlook (Forecast from day 3 to day 7)

The warnings bulletins are not available as voice file.

There is no history that is kept of the voice files. Every file is overwritten when an updated file is created.

The overarching README file of the datamart itself can be found here:
http://dd.weather.gc.ca/about_dd_apropos.txt

###############################################
###       Filename                          ###
###############################################
The provincial and territorial summary XML files are grouped in a directory tree that is 
based on the province/territory and the summary type.

1- The directories have the following nomenclature:
/citypage_weather/mp3/PC/

where:
* PC is a 2 letter provincial or territorial code. The code is one of the 13 following values:
** AB (Alberta)
** BC (British Columbia)
** MB (Manitoba)
** NB (New Brunswick)
** NL (Newfoundland and Labrador)
** NS (Nova Scotia)
** NT (Northwest Territories)
** NU (Nunavut)
** ON (Ontario)
** PE (Prince Edward Island)
** QC (Québec)
** SK (Saskatchewan)
** YT (Yukon)

2- The filenames have the following nomenclature:
 LOCATION_TYPE_L.mp3

where:
* LOCATION: Location code for this voice file. This code as the form 's0000NNN'. The complete locations list is available here under the directory http://dd.weather.gc.ca/citypage_weather/docs/
* TYPE: string indicating the type of the voice file. It can be one of those 3 types:
** 'sa' is for current weather condition
** 'd1d2' is for the short term forecast (Day 1 and 2)
** 'd3d7' is for the outlook (Forecast from day 3 to day 7)
* L: 1 letter indicating the language of the file. Can be one of the values:
** f (French)
** e (English)

Examples for Halifax:
 s0000318_sa_e.mp3
 s0000318_d1d2_e.mp3
 s0000318_d3d7_e.mp3

########################
###    Description   ###
########################

Ce document explique les fichiers voix en format mp3
distribués sur le datamart d'Environnement Canada à cet URL:
http://dd.meteo.gc.ca/citypage_weather/mp3/

Chaque fichier permet à l'usager d'entendre les prévisions météorologiques
pour différentes villes du Canada ou d'entendre les conditions
météorologiques actuelles.

Il y a 3 types de fichiers:
1- Fichier avec les conditions météorologiques actuelles
2- Fichier avec la prévision météorologique à court terme (Jour 1 et 2).
3- Fichier avec l'aperçu à long terme (Jour 3 à 7)

Les messages d'alertes et d'avertissement ne sont pas disponible en fichier vocal.

Il n'y a pas d'historique des fichiers vocaux sur le serveur. Chaque fichier est écrasé lorsqu'un nouveau fichier vocal est créé.

Le fichier README global pour le datamart peut être trouvé ici:
http://dd.meteo.gc.ca/about_dd_apropos.txt

##############################################
###       Noms de fichier                  ###
##############################################

Les fichiers vocaux sont regroupés dans une structure de répertoires basée sur la
province/territoire.

1- Les répertoires suivent cette nomenclature:
/citypage_weather/mp3/PC/

où:
* PC: code de 2 lettres pour la province ou le territoire. Ce code peut prendre une de ces 13 valeurs:
** AB (Alberta)
** BC (Colombie-Britannique)
** MB (Manitoba)
** NB (Nouveau-Brunswick)
** NL (Terre-Neuve-et-Labrador)
** NS (Nouvelle-Écosse)
** NT (Territoires du Nord-Ouest)
** NU (Nunavut)
** ON (Ontario)
** PE (Île-du-Prince-Édouard)
** QC (Québec)
** SK (Saskatchewan)
** YT (Yukon)


2- Les noms de fichiers suivant cette nomenclature:
 LOCATION_TYPE_LANGUAGE.mp3

où:
* LOCATION: Code d'endroit du fichier vocal. Le code à le format  's0000NNN'. La liste complète des endroits est disponible dans le répertoire http://dd.meteo.gc.ca/citypage_weather/docs/
* TYPE: chaîne de caractères indicant le type de fichier. Il prendre l'une de ces 3 valeurs:
** 'sa' pour les conditions météorologiques actuelles
** 'd1d2' pour la prévision météorologique à court terme (Jour 1 et 2).
** 'd3d7' pour l'aperçu à long terme (Jour 3 à 7)
* L: 1 lettre indiquant la langue du fichier. Peut-être une de ces valeurs:
** f (français)
** e (anglais)


Exemples pour Halifax:
 s0000318_sa_f.mp3
 s0000318_d1d2_f.mp3
 s0000318_d3d7_f.mp3
